# 🌌 Симулятор Рождения Вселенной (ЕТВП)

Интерактивная 3D-визуализация эволюции Ψ-поля от первичной флуктуации до формирования структур.

## 🎮 Управление
- **Клик** — создать первичную флуктуацию (рождение солитона).
- **ПРОБЕЛ** — удерживать для повышения когерентности (C) поля.
- **Мышь** — вращение камеры, зум, панорама.

## 📊 Термодинамика поля
- **C** — когерентность (мера порядка).
- **E** — энергия системы.
- **S** — энтропия (мера хаоса).
- **F = E - ΘS** — свободная энергия.

## 🔬 Физический смысл
- **Низкая C:** Хаос, высокая энтропия, разлёт солитонов (инфляция).
- **Высокая C:** Порядок, низкая энтропия, солитоны стягиваются к центру и синхронизируются.
- **Ритмичное нажатие ПРОБЕЛа:** Живое динамическое равновесие (Z-принцип).
