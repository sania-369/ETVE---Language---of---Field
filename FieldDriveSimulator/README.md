# 🚀 Полевой Двигатель (ЕТВП)

Интерактивная 3D-визуализация безреактивного движения на основе управления градиентом когерентности Ψ-поля.

## 🎮 Управление
- **ПРОБЕЛ** — удерживать для повышения когерентности (C).
- **Мышь** — вращение камеры, зум, панорама.

## 📊 Параметры
- **C** — когерентность поля.
- **Ускорение** — в единицах g.
- **Тяга** — удельная тяга в Н/кг.

## 🌀 Физический смысл
При повышении когерентности (внимание оператора) Ψ-поле вокруг аппарата становится **асимметричным**: оно смещается назад, создавая градиент. Аппарат «втягивается» в область с более высокой когерентностью, двигаясь **без отброса массы**. Это демонстрирует Z-принцип и полевую природу движения.

## 🔬 Связь с меморандумом
Симулятор визуализирует предсказания из инженерного меморандума ЕТВП по полевым двигателям:
- Рабочее тело: наноструктурированный никель.
- Резонансная частота: ~0.55 ТГц.
- Тяга растёт с когерентностью оператора (C > 0.7).
