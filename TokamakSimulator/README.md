# 🌀 Токамак Симулятор (ЕТВП)

Визуализация удержания плазмы в токамаке на основе принципов ЕТВП.

## 🎮 Управление
- **ПРОБЕЛ** — удерживать для повышения когерентности (C).
- **Мышь** — вращение камеры, зум.

## 📊 Параметры
- **C** — когерентность поля.
- **τ_E** — время удержания плазмы (растёт с C).

## 🔬 Физический смысл
При повышении когерентности (внимание оператора) плазма становится стабильнее, ярче, а время удержания увеличивается. Это демонстрирует Z-принцип и влияние наблюдателя на физическую систему.
