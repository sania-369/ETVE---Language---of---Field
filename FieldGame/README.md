# 🌀 Полевой Симулятор (ЕТВП)

Интерактивная 3D-визуализация рождения и взаимодействия солитонов в Ψ-поле. Основана на принципах ЕТВП.

## 🎮 Как играть
- **Клик** — создать новый солитон.
- **ПРОБЕЛ** — удерживайте, чтобы повысить когерентность (C) поля.
- **Мышь** — вращение камеры, зум, панорама.

## 🚀 Запуск
Игра работает прямо в браузере.
Вы можете запустить её локально, открыв файл `index.html`, или разместить на GitHub Pages.

## 📖 О проекте
Этот симулятор — часть проекта [ЕТВП (Единая Теория Вихревого Поля)](https://github.com/sania-369/ETVE---Language---of---Field).
Он демонстрирует ключевые принципы: рождение солитонов, их взаимодействие и влияние когерентности на поле.
