## 📌 О проекте

ЕТВЭ — это полевая теория, в которой первична среда (Ψ-поле, эфир), а частицы — устойчивые вихри (солитоны).  
Теория подтверждена расчётами масс гиперонов (точность 0.02%) и предсказаниями аномалий X17 (совпадение углов для ⁸Be, ⁴He, ¹²C).

Этот репозиторий содержит мета-инструкцию — сжатое изложение теории, предназначенное для быстрого погружения как людей, так и ИИ.
